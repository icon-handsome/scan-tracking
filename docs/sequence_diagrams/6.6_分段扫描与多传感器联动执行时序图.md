# 6.6 分段扫描与多传感器联动执行时序
对应文档章节：6.6 分段扫描与多传感器联动执行时序
适用场景：6.5 环节通过位姿考核放行开启多节点寻迹。

``mermaid
sequenceDiagram
    participant P as PLC
    participant I as IPC (2D/3D相机/跟踪定位)
    participant M as 机械臂/转台驱动

    P->>P: 编排本分段位置、通知转台驱动法向、指示跟踪与2D到位就绪
    P->>I: 下发当前段索引 ScanSegmentIndex

    P->>M: 驱动到目标位
    
    P->>I: 到位后触发 Trig_ScanSegment = 1
    
    I->>I: 捕捉侧沿
    I->>P: 回复 Ack_ScanSegment = 1
    
    I->>I: 驱动本段位姿下2D定向打光采集
    I->>I: 稍微延迟避光执行本段3D照面采集
    I->>I: 调用视觉跟踪坐标套系融合暂存
    
    I->>P: 本段保存即后推向 Ack_ScanSegment = 2
    
    alt 扫描正常
        P->>P: 得回后置零触发
        
        opt 有下一段序号剩余
            P->>M: 转向移动执行下一段扫描
        end
        opt 无下一段序号
            P->>P: 退出执行扫描归零泊车
        end
    else 条件不满足/采集失败 (Res_ScanSegment = 2/5)
        P->>P: 报相应错代码，拉响警告，请求干预
    end
``
