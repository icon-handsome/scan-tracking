# 6.7 综合 OCR 缺陷识别与 3D 解算时序
对应文档章节：6.7 综合 OCR 缺陷识别与 3D 解算时序
适用场景：上阶段扫全轮廓返回空档准备阶段。

``mermaid
sequenceDiagram
    participant P as PLC
    participant I as IPC (算法池)
    
    P->>P: 发送整体运算提报触发 Trig_Inspection = 1
    
    I->>I: 捕捉触发侧沿
    I->>P: 回复 Ack_Inspection = 1
    
    I->>I: 调集此前多套采集组，并发投给后发算法池
    I->>I: 2D进行AI表征编号OCR摘取、表面缺陷划识
    I->>I: 3D进行管口直径、高深厚度与坡角、接缝等计算汇演
    
    I->>I: 根据多阈值筛算得总评价
    
    alt 判定优良 (Res_Inspection=1)
        I->>I: 敲定判定结果
    else 判定次废 (Res_Inspection=2/3)
        I->>I: 附NG分项代码存入 NG_Reason_Word0/1
    end
        I->>I: 详细扩测数据(容积长度)填表
    
    I->>P: 再发交 Ack_Inspection = 2 放行终控端
    
    P->>P: 提取结果判断入档
    P->>I: 复0拉起源 Trig_Inspection = 0
``
