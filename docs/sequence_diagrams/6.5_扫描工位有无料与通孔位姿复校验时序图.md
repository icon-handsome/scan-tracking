# 6.5 扫描工位有无料与通孔位姿复校验
对应文档章节：6.5 扫描工位有无料与通孔位姿复校验
适用场景：毛坯刚落实到小转台工位，进入正式检测前哨。

``mermaid
sequenceDiagram
    participant P as PLC
    participant I as IPC (2D相机/扫描仪)
    participant O as 操作员
    
    P->>P: 将带云台2D相机/扫描仪开入观测基点1号位
    
    P->>I: 激活 Trig_StationMaterialCheck = 1
    
    I->>I: 拍照判断托盘是否压料
    
    alt 判断有料
        I->>P: 交验 Res=1 (有)，Ack=2
        
        P->>P: 运往定点观测点2
        
        P->>I: 激活 Trig_PoseCheck = 1
        
        I->>I: 寻边算法分析通孔特征
        I->>I: 确定夹持方势和位置约束
        
        I->>P: 填送偏差比回执 Res 答复
        
        alt Res=1 (姿态正确/可夹持)
            P->>I: 接收判定数据，双拉起触发复原
        else Res=3/位姿不可接纳
            P->>P: 发声光报案并禁止推进核心扫描段
            O->>O: 人介入手工校核吊件或重新退磁卸走
        end
        
    else 判断无料
        I->>P: 交验 Res=2 (无料)，Ack=2
        
        P->>P: 发声光报案并禁止推进核心扫描段
        O->>O: 人介入手工排查
    end
``
