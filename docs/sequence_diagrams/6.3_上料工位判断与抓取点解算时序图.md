# 6.3 上料工位判断与抓取点解算时序
对应文档章节：6.3 上料工位判断与抓取点解算时序
适用场景：已按下自动启动并具备放料准备，安全检测全部通过。

``mermaid
sequenceDiagram
    participant P as PLC
    participant I as IPC (顶架3D相机)
    
    P->>P: 判断探测器，确认上料口备好待传元件
    P->>P: 扫描工位清空无占坑
    
    P->>P: 写入任务号、配方号、产品类型
    P->>I: 触发抓取点请求 Trig_LoadGrasp = 1
    
    I->>I: 捕捉触发侧沿
    I->>P: 落盘 Ack_LoadGrasp = 1
    
    I->>I: 指挥顶架3D相机执行拍照点云构建
    I->>I: 解算提取6轴吸取基准法向
    
    alt 解算正常
        I->>P: 送入 Load_X~Load_Rz 坐标缓冲组
        I->>P: 设置 Res_LoadGrasp = 1 (成功)
        I->>P: 推进握手 Ack_LoadGrasp = 2
        
        P->>P: 判断 Ack=2
        P->>P: 提取各轴位置
        P->>I: 清零起始触发位 Trig_LoadGrasp = 0
        
        I->>I: 侦知 Trig=0
        I->>P: 回零 Ack_LoadGrasp = 0，重置状态机
    else 3D审视结论为无料或超出抓取阈值 (超时/错误)
        I->>P: 报相应错代码 (Res_LoadGrasp = 3/4等)
        I->>P: 推进握手 Ack_LoadGrasp = 2
        
        P->>P: 停止进给，拉响黄色告铃
        P->>P: 请求人为接入复位判定
    end
``
