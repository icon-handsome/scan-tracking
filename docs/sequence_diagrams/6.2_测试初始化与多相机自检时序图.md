# 6.2 测试初始化与多相机自检
对应文档章节：6.2 测试初始化与多相机自检
适用场景：人工上位机交互发下自检指令（且机械臂等位于原点安全位）。

``mermaid
sequenceDiagram
    participant O as 操作员/上位机
    participant P as PLC
    participant I as IPC

    O->>P: 发起自检指令
    P->>P: 检查机械臂等是否位于原点安全位
    
    P->>I: 下发自检指令 (Trig_SelfCheck=1)
    
    I->>I: 巡调 3D 相机外参
    I->>I: 巡调 2D 解算
    I->>I: 跟踪标定校验项
    
    I->>P: 返回自检结果 (Ack_SelfCheck, Device_Fault某位域)
    
    alt 自检超阈值 (NG)
        P->>P: 触发红色类故障，记录日志
        P->>P: 禁止运行，提醒运维介入操作
        O->>O: 运维人员介入
    else 自检合格 (OK)
        P->>P: 开启允许自动运行位 (Safety_Status_Word.Bit7=1)
    end
``
