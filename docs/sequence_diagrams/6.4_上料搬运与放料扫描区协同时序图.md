# 6.4 上料搬运与放料扫描区协同
对应文档章节：6.4 上料搬运与放料扫描区协同
适用场景：PLC 顺利拿到抓取解算坐标位，进入搬动周期。

``mermaid
sequenceDiagram
    participant P as PLC
    participant O as 操作员/吊带
    participant M as 机械臂

    P->>P: 根据目标做两步靠近抓取控制
    alt (接触锁定)
        P->>M: 机械臂抵近 -> 下向正方接触，充磁锁料
    end
    
    P->>M: 机械臂起跳携物移向扫描工作转面区
    
    P->>M: 进行两步精准卸料投下
    alt (到达/贴脱磁)
        P->>M: 到达正上方 -> 贴贴退磁
    end
        P->>P: 放料完结
        P->>P: 上下料机清让并回到待命泊车点
    
    opt 人为中断或停电掉链失磁
        P->>P: 废除此件周期
        O->>O: 使用吊带运回上料点后再执行系统重置
        P->>P: 清空数据复查
    end
``
