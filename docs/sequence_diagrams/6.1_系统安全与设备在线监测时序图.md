# 6.1 系统安全与设备在线监测时序

对应文档章节：6.1 系统安全与设备在线监测时序
适用场景：系统上电，循环执行（周期建议 50~100ms）。用于维持两端通信状态及外围安全。

``mermaid
sequenceDiagram
    participant P as PLC
    participant I as IPC (安全模块)
    
    loop 循环（周期 50~100ms）
        P->>P: 刷新硬件安全单元（围栏门锁、光栅等）
        I->>I: 24h不间断录像识别，判断人员进入
        opt 有人员进入
            I->>P: 写入 Safety_Status_Word (上报禁行位组合)
        end
        I->>P: 回传吊装防碰和内壁防碰雷达信号
        I->>P: 发起 PLC_Heartbeat 周期心跳信号
        
        alt 正常情况
            P->>I: 同步自身对应的 IPC 反馈端状态
        else 断联情况 (PLC 检测到3秒无跳变)
            P->>P: 判定掉联，触发红色预警级别
            P->>P: 锁停主要动设备电源
            P->>P: 发出声光报警，拦截自动状态机推进
            I->>I: 软件持续弹窗直至排故复位
        else 人员侵入/碰撞预警
            P->>P: 判定触发红色预警级别
            P->>P: 锁停主要动设备电源
            P->>P: 发出声光报警，拦截自动状态机推进
            I->>I: 软件持续弹窗直至排故复位
        end
    end
``
